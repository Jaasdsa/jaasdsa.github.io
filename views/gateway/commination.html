# 设备通信

# 通讯方式

  • 串口通信 串行接口是一种可以将接收来自CPU的并行数据字符转换为连续的串行数据流发送出去，同时可将接收的串行数据流转换为并行的数据字符供给CPU的器件。一般完成这种功能的电路，我们称为串行接口电路。
  • 以太网通信 以太网是一种基带局域网技术，以太网通信是一种使用同轴电缆作为网络媒体，采用载波多路访问和冲突检测机制的通信方式，数据传输速率达到1Gbit/s，可满足非持续性网络数据传输的需要。

# 环境准备

  • kepserver6.4
  • kepserver6.4补丁
  • jre-8u241-windows-i586
  • mqtt.fx
  • emq

# KepServer Modbus RTU Serial

  1. kepServer添加通道选择Modbus RTU Serial驱动
  2. 指定通道名称
  3. 分配通道
  4. **重要**选择COM口
  5. 选择完成添加通道
  6. 添加设备
  7. 确定设备型号
  8. **重要**确定从站ID
  9. 选择完成添加设备
  10. **重要**根据地址表添加条目
  11. 验证采集结构，通信质量良好证明通信成功

# KepServer IoT Gateway

使用KepServer IoT Gateway插件需要安装32位jrejre-8u241-windows-i586

  1. 添加IoT Agent,选择MQTT Client
  2. 填写MQTT服务器参数信息 我这里使用的EMQ服务器搭建的测试服务器，可以参考emq
  3. 填写MQTT客户端认证信息,注意:client ID相同会踢掉前任
  4. 选择要转发的条目
  5. 转发规则
    • only Data Changes数据变化时上报
    • every scan每次轮询就上报
  6. 验证转发，这里采用的MQTT.fx订阅mqtt服务器数据mqtt.fx
  7. 控制主题,网关订阅主题实现反向控制设备
